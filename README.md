[Socket-Chat-App](https://youtu.be/wNS29d8GB0s)

## `How to use`

* Use follow command for cloning app
  ~~~
  git clone https://github.com/PratapTyagi/socket-chat-app.git
  ~~~
  
* Open command prompt
* use
  - For backend
  ~~~
  npm i
  nodemon run dev
  ~~~
  - For front end
  ~~~
  cd client
  npm i
  npm start
  ~~~
